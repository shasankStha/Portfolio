Webpage Desigining

The process of creating a web pages appearance and to the choice of a right color scheme, page layout, fonts and more is called web page desigining. It usually refers to the user experience aspects of website development rather than software development.

Web design used to be focused on designing websites for desktop browsers only but nowadays design for mobile and tablet browsers has also become ever-increasingly important.

HTML

The HTML is the standard markup language for desigining documents to be displayed in a web browser. It can be assisted by technoligies such as CSS and Javascript 

    example:
    < !DOCTYPE html >
    < html >
    < head >
        < title >Document< /title >
    < /head >
    < body >

        < p >This is paragraph.< /p >
    < /body >
    < /html >
