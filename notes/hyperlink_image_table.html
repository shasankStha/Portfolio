Hyperlink

HTML links are hyperlinks. We can click on a link and jump to another document.

Syntax:
The HTML < a > tag defines a hyperlink. It has the following syntax:
< a href="url"> link text < /a >

Target Attribute

The target attribute specifies where to open the linked document.

The target attribute can have one of the following values:
_self - Default. Opens the document in the same window/tab as it was clicked
_blank - Opens the document in a new window or tab
_parent - Opens the document in the parent frame
_top - Opens the document in the full body of the window

Image

The HTML < img > tag is used to embed an image in a web page.

Syntax:
< img src="url" alt="alternative text">

Attributes

"alt" attribute - Provides alternative text instead of image when image cannot be load.
"width" attribute - Define the width of image.
"height" attribute - Define the height of image.

Table

HTML tables allow web developers to arrange data into rows and columns.A table in HTML consists of table cells inside rows and columns.

The HTML < table > is used to define table.
Each table cell is defined by a < td > and a < /td > tag.
Each table row starts with a < tr> and end with a < /tr> tag.

Sometimes you want your cells to be headers, in those cases use the < th> tag instead of the < td> tag.
