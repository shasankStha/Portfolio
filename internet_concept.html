ISP
Internet Service Provider(ISP) is a company thet manages some special routers that are all linked together and can also access other ISPs' router.

IP address
Any computer linked to a network has an unique address thet identifies it, called IP address. eg 192.168.100.10. Remembering IP addresses is hard for human mind, so we alias an IP address within ahuman redable name called domain name.

Intranets
Intranets are the private networks that are restricted to members of a particular organization.

Extranets
Extranets are the part of a private network to allow sharing and collaboration with other organizations.

Webpage
A document which can be displayed in a web browser such as Firefox, Google, Safari etc. these are called web pages.

Website
A collection of webpage which are group together and usually connected together in various ways often called a website or site.

Webserver
A computer thet hosts a website on the internet is called webserver.

Search engine
A web service that helps you to find other web pages is called search engine. They are often accessed through web browser.

HTTP
Hypertext Transfer Protocol(HTTP) is a protocol for fetching resources such as HTML documents. It is Client server protocol and Application layer protocol.

WWW
The World Wide Web is the world's document software platform. It is an information space where documents and other web resources can be accessed through the Internet using a webbrowser.

Note: Internet and the web are two different things. The internet is an infrastructure wheras the web is a service built on top of infrastructure.

URI
Uniform Resources Identifier(URI) is the sequence of characters that distinguishes one resource from another. A URI must have a scheme name. eg.
foo://example.com:8042/over/there?name=ferret#nose
telnet://192.0.2.16:80/
foo: Scheme name
example.com: Authority

There are two types of URI

  1. Uniform Resources Name(URN)
  2. Uniform Resources Locator(URL)

URL
Uniform Resources Locator is a specific type of identifier that not only identifies the resource but tells you how to access if or where it is located. eg.
ftp://ds.internic.net/internet-drafts
ftp: Protocol
internic.net: authority
